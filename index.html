tutorial point cloud from depth images

20 diciembre 2020

POINT CLOUD FROM PROFILING LASER IMAGES

6 enero de 2021

Aquí se muestra un procedimiento para hacer una reconstrucción 3D a partir de imagenes 2D de perfilación laser. Hay un total de 59 imágenes, se calculan las coordeandas u,v del perfil en la imágen y luego se crea la coordenada z, finalmente se crea la nube de puntos 3D y se visualiza

Tecnologías: Python, Vedo, Numpy, OpenCV, camara perfilador laser, git, github

Deptherm-Inspection

20 Febrero de 2021

Deptherm es una aplicación para inspección termográfica. Ofrece herramientas como manejo de usuarios, calibración de cámaras, fusión de imágenes rgb-térmica, rgb-profundidad, térmica-profundidad, rgb-térmica-profundidad. Permite visualización 2D y 3D.

FuchiCovid

20 noviembre de 2020

FuchiCovid es una aplicación diseñada para el registro y seguimiento de pacientes diagnosticados con el virus COVID-19, la aplicación permite el manejo de 2 tipos de usuarios: funcionaros, encargados de registrar médicos y pacientes. Y médicos, encargados de visitar y registrar información de los pacientes. Los dos pueden ver un reporte general

Tecnologías: Node.js, Vue.js, Express, PostgresSQL, Heroku
